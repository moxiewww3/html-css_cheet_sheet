HTML Element Reference

Main Root

Element Description
<html> The root (top-level) Element of an HTML document. All other elements must be desecendants of this element.

Sectioning Root

Element Description
<head> Contains metadata/information about the html document.
<body> Represents the content of an HTML document.

Document metadata

Element Description
<title> The <title> defines the document's title and is shown in a Browser's title bar or page's tab.
<link> The <link> element specifies relationships betwween the current document and an external resource. Commonly used to link to CSS.
<style> The <style> contains style information for a document. It contains CSS, which is applied to the contents of the documents containing the <style> element.
<meta> The <meta> element represents Metadata that cannot be represented by other HTML meta-related elements, like base, link, script.
